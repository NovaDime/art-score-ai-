# AI智能评估与成长系统

## 项目简介

AI智能评估与成长系统是一个基于 **DGX Spark** 与本地多模态模型构建的图片作品智能理解与评估系统。  
系统支持教师输入评分标准，学生上传图片作品后，自动完成：

- 图片内容识别与描述
- 五维量化评分
- 综合评价
- 学习建议
- 雷达图可视化展示
- 历史记录保存

本项目重点不在于单纯调用大模型，而在于将多模态模型输出转化为 **结构化、可解释、可展示** 的评估结果，形成完整的前后端产品闭环。

---

## 项目背景

在学校美术、设计、图片类作业评估场景中，教师往往需要花费大量时间进行重复性批改，同时学生获得的反馈也常常不够统一和结构化。

传统问题主要包括：

- 教师批改耗时较长
- 评价标准不够统一
- 学生获得的反馈不够细致
- 图片类作业难以快速形成量化分析结果

因此，我们希望构建一个本地部署的 AI 评估系统，让模型先理解图片内容，再根据教师提供的评分标准，输出结构化评分结果与改进建议，从而提升评估效率和反馈质量。

---

## 项目目标

本项目面向图片作品上传场景，构建一个可以本地运行的多模态 AI 评估系统。  
用户上传图片并输入评分标准后，系统自动完成图片理解、五维评分、综合评价、学习建议与可视化展示，形成完整评估闭环。

---

## 核心功能

### 1. 图片上传与预览
支持用户上传 JPG / JPEG / PNG / WEBP 图片，并在前端实时预览。

### 2. 教师自定义评分标准
教师可以在前端输入评分 rubric，模型将结合 rubric 进行评估。

### 3. AI 图片内容识别
系统会先对图片进行视觉理解，生成自然语言图片描述，而不是直接模板化评分。

### 4. 五维量化评分
从以下五个维度进行 0-100 分量化评分：

- composition（构图）
- color（色彩）
- creativity（创意）
- completeness（完整度）
- theme（主题契合）

### 5. 综合评价与学习建议
系统自动生成：
- 综合评价
- 学习建议
- 核心优势
- 待提升点

### 6. 雷达图可视化展示
前端将五维评分结果实时转化为雷达图，增强展示效果与结果解释性。

### 7. 历史记录保存
后端支持保存评估历史，便于后续扩展成长轨迹分析。

## 技术创新点

本项目的创新点不在于单纯使用大模型，而在于将多模态模型输出进一步转化为 **结构化评估系统**。

具体体现在以下几点：

### 1. 从“自然语言输出”升级为“结构化评分结果”
系统不是简单生成一段评语，而是将模型输出组织为：

- 图片内容描述
- 五维评分
- 综合评价
- 学习建议
- 优势与不足

这使得模型输出可以直接驱动前端展示，而不是停留在文本层。

### 2. 构建了完整的产品闭环
项目已经完成：

- 前端页面展示
- FastAPI 后端服务
- 本地多模态模型调用
- 结果结构化解析
- 可视化雷达图展示

形成了一个可以运行、可以演示、可以提交的完整系统，而不是静态网页或单点脚本。

### 3. 先“看懂图”，再“给评分”
系统在评分前先生成图片内容理解结果，增强了评估的可解释性，避免模型看起来像“直接套模板打分”。

### 4. 本地部署，贴合 DGX Spark 使用场景
项目采用本地模型部署方式，减少对云端依赖，更符合 DGX Spark 强调的本地推理、低延迟和可持续运行场景。

## 场景落地价值

本项目面向真实教学场景，适用于：

- 高校美术作业评估
- 设计类课程作品点评
- 校园艺术作品初步筛选
- 学生自评与练习反馈

其落地价值主要体现在：

### 对教师
- 减少重复性批改劳动
- 提供统一的基础评分框架
- 提升批改效率

### 对学生
- 获得更结构化的反馈
- 更容易理解自己的优势与不足
- 提高自主改进能力

### 对校园部署场景
- 采用本地部署方式
- 不依赖外部云端服务
- 更有利于作品数据与隐私保护

因此，本项目不仅是一个演示系统，也具备进一步校园化部署与扩展的潜力。
