# AI智能评估与成长系统 项目报告书

## 一、 项目背景与目标
在教学场景中，美术、设计、摄影等作品的人工评分长期存在评价标准不一、反馈单一且教师工作负担过重等痛点。本系统旨在通过部署本地多模态大模型，构建一个包含图像内容理解、多维度量化评分、自动生成可解释反馈及可视化展示的完整工作流。核心目标是将传统的“单向打分工具”升级为“AI 驱动的成长辅助系统”。

## 二、 核心功能架构
系统已实现端到端的功能闭环：
* **多模态输入与解析**：支持学生上传图片作品，系统同时解析图像特征与教师录入的文本评分标准。
* **五维结构化评价**：打破传统单一给分机制，强制输出基于构图、色彩、创意、完整度、主题表达五个维度的量化分数。
* **闭环反馈引擎**：提供结构化的 JSON 结果，包含底层视觉内容描述、优劣势定位及改进建议。
* **数据可视化与沉淀**：前端实时渲染五维雷达图与综合得分，后端记录时间序列上的评估历史，建立学生能力成长轨迹。

## 三、 技术架构与双轨演进路线（核心技术点）
系统基于 DGX Spark 算力底座，采用 FastAPI 后端与 HTML/JS 前端分离架构。为兼顾比赛验收的绝对稳定性与对前沿架构的性能压榨，系统采用双轨开发模式：

### 1. 稳定版 V7.4（本次视频演示采用版本）
* 采用经过优化的轻量级本地多模态大模型，跑通了识别、解析、结构化 JSON 输出与前端渲染的全链路。
* 在 DGX Spark 环境下并发响应极速，不存在显存溢出或网络阻断风险，数据100%不出局域网，满足严苛的隐私安全合规要求。

### 2. 高阶实验版 V8.0（前沿性能压榨与架构演练）
* **72B 满血视脑接入**：全面舍弃轻量模型，底层调度满血版千问 72B（Qwen-VL-72B）垂类模型主导图像深度理解与大师级评语书写。极致利用 DGX Spark 的 128GB 大统一内存，对复杂构图进行细粒度解构。
* **RAG 标准量化引擎**：引入检索增强生成（RAG）知识库，将教师输入的非结构化评判要求转化为向量。大模型在推理前先进行向量检索，对主观审美标准进行严密的数学量化与精准定位，进一步消除模型幻觉。
* **云边异构路由**：在本地 Spark 节点之外，预留了对接 GPT 等云端高阶 API 的旁路网关。设计了“本地处理高密隐私数据 + 云端兜底复杂逻辑”的拓扑。因目前该路由机制存在偶发性超时，尚未合入主干稳定版本，故未在最终演示视频中展现。

## 四、 场景落地与商业延伸
本方案实现了从“单纯视觉识别”到“强干预教学评估”的跨越。
* **私有化教育基站（核心盘）**：无缝下沉至高校设计学院或艺术培训机构，接管美术、摄影作业的高频初筛与基础批阅。本地化部署方案彻底打消校方对学生创意资产外泄的顾虑。
* **数字资产自动化 QA**：扩展应用于游戏外包品控或 AIGC 内容审核流水线，精准检测人物比例、色彩合规性等是否满足设计白皮书标准，大幅削减主美人工复核成本。

## 五、 演示效果说明
为保证评审体验的流畅度，提交的演示录屏基于稳定版 V17.4 录制。视频完整展现了教师标准录入、学生作品极速上传、多模态实时评估以及五维雷达图的动态生成过程，逻辑严谨，可完美复现。
